DB

- DB 생성
docker run --rm -d --name mjDB \
-p 3306:3306 \
-e MYSQL_DATABASE=mjDB \
-e MYSQL_ROOT_PASSWORD=password \
mysql:5.7 \
--character-set-server=utf8 \
--collation-server=utf8_general_ci

- DB 런
docker exec -it mjDB  mysql -u root -p  (password = password)

- DB 관리
1. flask db heads : 현재 migrate 작업의 최종 리비전을 확인한다. 
2. flask db current : migrate 작업의 현재 리비전을 확인한다. 
3. flask db stamp heads : 현재 리비전을 최종 리비전으로 변경한다.
